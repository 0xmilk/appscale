PbServer

To support the Google App Engine Datastore API in a way that is database-agnostic, App Engine applications store and retrieve data via the PBServer. The server inherits this name from the storage format of requests in the Datastore API: Protocol Buffers.

Constants

NUM_PBSERVERS

As PBServers are single-threaded, we run more than one and put nginx in front of it to load balance requests. This constant indicates how many PBServers should be run on each node.

STARTING_PORT

The first port that should be used to host PBServers.

PROXY_PORT

The port that we should run nginx on, to load balance requests to the various PBServers running on this node.

LISTEN_PORT_NO_SSL

The port that nginx should be listening to for non-encrypted requests to the PBServers.

LISTEN_PORT_WITH_SSL

The port that nginx should be listening to for encrypted requests to the PBServers.

DBS_NEEDING_ONE_PBSERVER

A list of databases that we cannot spawn multiple PBServers for and stick nginx in front of, usually because they have locks or other database state stored in memory.

DBS_WITH_NATIVE_PBSERVER

A list of databases that we have native (not database agnostic) support for within AppScale.

NAME

The name that nginx should use as the identifier for the PBServer when it we write its configuration files.

def self.get_executable_name(table)
  if DBS_WITH_NATIVE_PBSERVER.include?(table)
    return "#{APPSCALE_HOME}/AppDB/appscale_server_#{table}.py"
  else
    return "#{APPSCALE_HOME}/AppDB/appscale_server.py"
  end
end

def self.get_server_ports(table)
  if DBS_NEEDING_ONE_PBSERVER.include?(table)
    num_pbservers = 1
  else
    num_pbservers = NUM_PBSERVERS
  end

  server_ports = []
  num_pbservers.times { |i|
    server_ports << STARTING_PORT + i
  }
  return server_ports
end

Starts a Protocol Buffer Server on this machine. We don’t want to monitor it ourselves, so just tell god to start it and watch it.

# File lib/pbserver.rb, line 58
def self.start(master_ip, db_local_ip, my_ip, table, zklocations)
  pbserver = self.get_executable_name(table)
  ports = self.get_server_ports(table)

  env_vars = { 
    'APPSCALE_HOME' => APPSCALE_HOME,
    "MASTER_IP" => master_ip, 
    "LOCAL_DB_IP" => db_local_ip 
  }

  ports.each { |port|
    start_cmd = "/usr/bin/python2.6 #{pbserver} -p #{port} " +
        "--no_encryption --type #{table} -z \'#{zklocations}\' " +
        "-s #{HelperFunctions.get_secret()} -a #{my_ip} --key"
    stop_cmd = "pkill -9 appscale_server"
    GodInterface.start(:pbserver, start_cmd, stop_cmd, port, env_vars)
  }
end
